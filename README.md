# deterministic-design

**deterministic-design** is a Claude Code skill that renders your UI and *proves* it's balanced and usable — a **deterministic layout audit** (centroid / optical-center / pixel-oracle balance) plus a **vision-judged Nielsen usability audit** by a separate fresh-eyes judge. Determinism over the model's eye: numbers, not vibes.

> **What this is:** Improves existing design skills (including the default) by adding the layer they lack — it doesn't just advise on taste, it renders, measures, and judges the output.

## Install (Claude Code plugin)

```
/plugin marketplace add connerkward/deterministic-design-skill
/plugin install deterministic-design@deterministic-design-skill
```

Or install the whole set: `/plugin marketplace add connerkward/connerkward-skills`. Or drop this repo's `SKILL.md` into your agent's skills directory.
